# 番茄钟桌面应用 — Swing 开发计划

**依据**：`docs/prd/prd-4-28.md`（PRD v0.1）  
**UI**：Swing（可配合 FlatLaf 等主题提升观感）  
**说明**：Swing 能实现 PRD 要求的「小窗、低密度、清晰交互」；复杂动效与原生质感需更多自绘或后续评估 JavaFX/Compose。

---

## 实现前锁定（影响第 2～6 步）

1. **放弃番茄**：是否写入「番茄记录」且 `结果 = aborted`（PRD 推荐写入，便于统计中断率）。  
2. **暂停**：MVP **不支持暂停**（与 PRD 建议一致）。  
3. **飞书鉴权**：PAT 自用 vs OAuth（在第四步前确定）。

下文假设：**放弃写入 `aborted`**、**无暂停**、第四步前确定 PAT 或 OAuth。

---

## 第一步：工程骨架 + Swing 主窗体 +「假计时」闭环

**目标**：可运行桌面壳；不连飞书；用短时长验证布局与计时。

**内容**

- Gradle/Maven 工程；Java 21+。  
- 主窗：剩余时间大字号、进度条占位、「开始」「放弃」（放弃二次 `JOptionPane` 确认）。  
- 计时：`javax.swing.Timer` 或 `ScheduledExecutorService`，UI 更新走 `SwingUtilities.invokeLater`。  
- 状态机：`IDLE` → `RUNNING` → `ENDED` / `ABORTED`（结束可先占位对话框）。

**验收标准**

- 启动后主窗稳定，无未捕获异常。  
- 「开始」后倒计时正确走到 0；UI 持续刷新。  
- 「放弃」经确认后停止计时。  
- UI 更新在 EDT 上执行（长时间运行无明显假死）。

---

## 第二步：真实时长策略 + 系统通知 + 菜单栏（托盘）

**目标**：对齐 PRD 4.1 计时、通知、托盘。

**内容**

- 默认 25 分钟；时长存本地 `Preferences`（密钥仍后置）。  
- 正常结束：系统通知（实现方式在技术方案中写明；必要时 `osascript` 等）。  
- 可选短提示音 + 开关。  
- `TrayIcon` + `PopupMenu`：显示主窗、退出；tooltip 或菜单展示剩余时间或状态。  
- 可选：隐藏 Dock 图标（非硬性）。

**验收标准**

- 完整流程结束能收到系统级通知（开发可用 1 分钟等调试时长）。  
- 关主窗后可通过托盘恢复；退出后进程结束。  
- 声音开关生效。

---

## 第三步：休眠/合盖语义

**目标**：PRD 6.5 — 休眠期间不静默消耗番茄时间。

**内容**

- 休眠/唤醒检测（JNA/Cocoa 或「时间差」简化方案，局限写入技术方案）。  
- 休眠中暂停倒计时；唤醒后提示「休眠已暂停计时」+「继续」/「放弃」。

**验收标准**

- 休眠/合盖后剩余时间不再减少；唤醒后需用户确认才继续。  
- 无「休眠期间番茄偷偷跑完」。

---

## 第四步：飞书集成（只读）— 鉴权、配置、任务列表

**目标**：PRD 4.3 F-01～F-03、5.2 只读。

**内容**

- 配置：Base ID、表 ID、字段映射；首启向导（`JDialog` / `CardLayout`）。  
- HTTP：限流、指数退避、可读错误文案。  
- 启动拉取 +「刷新任务」；客户端过滤非已完成 + 标题搜索。  
- Token：macOS 钥匙串存储，不明文落配置文件。

**验收标准**

- 有效 Token 下列表与飞书抽样一致。  
- 断网/错误 Token 有提示、不崩溃。  
- 用户可读配置中无明文密钥。

---

## 第五步：结束卡片 + 写「番茄记录」+ 更新任务状态

**目标**：PRD 3.2、4.2、5.1、5.2。

**内容**

- 正常结束弹出模态结束卡片：备注（≤2000 字）、任务单选（可空）、有关联时可改状态（与飞书枚举一致）。  
- 保存：新增番茄记录（起止时间、计划时长、`completed`/`aborted`、备注、关联、`client_uuid`）。  
- 关联且状态变更时更新任务表。  
- 保存成功：轻量 Toast 或短暂提示。

**验收标准**

- 与 PRD §11 前三条一致：通知、番茄表新增正确、任务表状态可更新。  
- 未关联任务时状态控件不可用且不写任务表。

---

## 第六步：离线队列 + 幂等 + 重试

**目标**：PRD 6.3、5.4、§11 断网验收。

**内容**

- SQLite（或等价）持久化待同步队列；项含 `client_uuid` 与 payload。  
- 断网入队；联网后台重试；**同一 UUID 不重复插入**（应用层去重，文档说明飞书侧约束）。  
- 极简「重试」或失败列表入口。

**验收标准**

- 断网保存 → 入队；联网后自动同步，飞书仅一行、内容正确。  
- 重复提交同一 UUID 不产生重复业务行（按既定去重策略验证）。

---

## 第七步：设置/关于 + jpackage 分发 + 清单收尾

**目标**：PRD 4.4、6.2、§11 安装与密钥项。

**内容**

- 设置：飞书配置、默认时长、通知/声音；关于：版本、协议。  
- `jpackage` 产出 macOS 安装包；文档说明无 JDK 安装步骤。  
- 日志脱敏；主要路径 Tab 顺序合理（完整 VO 可 P1）。

**验收标准**

- 无 JDK 的 macOS 可安装并启动。  
- PRD §11 MVP 验收清单全部满足。  
- 用户文档写明：放弃策略、休眠行为、Token 轮换（若 PAT）。

## Swing 观感简要建议

- FlatLaf 或系统 L&F + 统一 `EmptyBorder`。  
- 大字号倒计时 + `JProgressBar` 或自绘圆角进度条。  
- 结束卡片固定小尺寸、模态，减少「灰旧 Swing」观感。
